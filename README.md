# LawNet

**Moteur de Raisonnement Algébrique par Manifold Topologique**

Une architecture de calcul neuronal qui traite les opérations arithmétiques comme des translations de phase dans un espace vectoriel continu.

## Structure du Projet

```
lawnet/
├── src/            # Code source du modèle et de la logique
├── tests/          # Tests de raisonnement (extensifs)
├── docs/           # Documentation détaillée
├── main.py         # Point d'entrée principal
└── requirements.txt
```

## Installation

1. Clonez le dépôt
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation

Pour lancer l'entraînement et les tests :

```bash
python main.py
```
